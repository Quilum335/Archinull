# Hyprland Dotfiles

A set of dotfiles and scripts for configuring a personalized Arch Linux environment with Hyprland. This setup includes:
- **Shell**: zsh
- **Terminal**: Foot
- **Application Launcher**: Rofi
- **Window Manager**: Hyprland
- **Notification Daemon**: Mako
- **Display Manager**: Ly
- **Bootloader**: REFInd
