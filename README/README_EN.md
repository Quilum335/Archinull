# Hyprland Dotfiles (English)

[Read this in Russian](README_RU.md)

This repository provides configuration files and setup scripts for Arch Linux with:
- **Shell**: zsh
- **Terminal**: Foot
- **Window Manager**: Hyprland
- **Notification Daemon**: Mako
- **App Launcher**: Rofi
- **Display Manager**: Ly
- **Bootloader**: REFInd
